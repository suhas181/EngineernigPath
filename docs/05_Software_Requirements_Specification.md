# 2. Product Overview

EngineerPath is an AI-powered platform designed to guide engineering students throughout their academic and career journey.

The platform combines career planning, personalized learning roadmaps, internship discovery, resume analysis, interview preparation, skill tracking, and AI mentorship into a single application.

Instead of relying on multiple websites, students can manage their entire learning journey from first year to placement through EngineerPath.

---

# 3. Business Goals

The primary business goals are:

- Simplify career preparation.
- Increase internship success.
- Improve placement outcomes.
- Reduce confusion among engineering students.
- Provide personalized AI guidance.
- Create a single platform for engineering career development.
- Build a scalable platform that supports millions of students.

# 4. User Roles

## Student

The primary user of EngineerPath.

Permissions:

- Create account
- Login
- Update profile
- Access roadmaps
- Upload resume
- Apply for internships
- Track progress
- Chat with AI Mentor
- Receive notifications

---

## Admin

Responsible for managing the platform.

Permissions:

- Manage users
- Add internships
- Verify resources
- Publish announcements
- View analytics
- Manage reports
- Moderate content

---

## Recruiter (Future)

Permissions:

- Post internships
- View applicants
- Manage job postings

---

## Mentor (Future)

Permissions:

- Publish learning content
- Conduct mentorship sessions
- Answer student questions
