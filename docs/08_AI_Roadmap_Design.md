# EngineerPath AI Roadmap Design

## Vision

The AI Roadmap is the core feature of EngineerPath.

It should not generate a generic roadmap.

It should create a completely personalized placement roadmap based on the student's current profile, skills, goals, and career target.

The roadmap should guide the student from their current level to becoming placement-ready in approximately 6–8 months.

---

# Goal

Help engineering students know exactly:

- What to learn
- When to learn it
- Why they should learn it
- Which resources to follow
- Which projects to build
- How much progress they have made
- How close they are to getting placed
